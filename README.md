# Pokemon World
    
Pokemon World is a web page with three main components: Generations, Locations and Pokemons. Where each component reads and displays detailed information taken from the Pok√©mon API, [PokeAPI](https://pokeapi.co/)

![pokemonworld](https://user-images.githubusercontent.com/50857082/124698204-f3726100-dead-11eb-80bb-9069e566f95b.gif)


## Installation

  1. Clone repository.
### `git clone https://github.com/deivid-01/PokemonWorld`

  2. Install npm packages.
### `npm install`

3. Run proyect
### `npm run start`
4. Open browser in [http://localhost:3000](http://localhost:3000)

## Deploy
Deploys the app for production to the `build` folder.\
for the best performance.
### `npm run deploy`
